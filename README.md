# Сравнение файлов

Приложение на Python (TkInter) для поиска строк, которые есть в **File_2**, но отсутствуют в **File_1**.

## Возможности

- Выбор файлов через стандартные диалоги Windows
- Чтение файлов в кодировке UTF-8
- Поиск уникальных строк в File_2 относительно File_1
- Read-only окно с возможностью копирования текста (Ctrl+C / контекстное меню)
- Сохранение результата в File_3
- Ярлык на рабочем столе для быстрого запуска

## Файлы

| Файл | Описание |
|---|---|
| `text_comparer.py` | Основное приложение |
| `create_shortcut.ps1` | Скрипт для создания ярлыка на рабочем столе |

## Запуск

```bash
python text_comparer.py
```

Или через ярлык **«Сравнение файлов»** на рабочем столе (создаётся скриптом `create_shortcut.ps1`).

## Использование

1. Нажмите **«Открыть»** — выберите File_1 (эталон), затем File_2 (проверяемый)
2. Просмотрите строки из File_2, которых нет в File_1
3. Нажмите **«Сохранить»** — сохраните результат в File_3
4. **«Закрыть»** — выход из приложения
