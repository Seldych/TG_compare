# TG Proxy — отбор прокси

Приложение на Python (TkInter) для отбора новых прокси-серверов Telegram.

**Задача:** есть старый список прокси (File_1) и новый список (File_2).
Приложение находит записи из нового списка, которых ещё нет в старом,
позволяет просмотреть их параметры (server/port/secret), удалить лишние
и сохранить актуальный результат.

## Возможности

- Сравнение двух списков proxy-URL через разность множеств
- Два метода проверки прокси: быстрый (TCP/TLS сокет) и точный (Telethon)
- Автоопределение TLS-режима по первому байту секрета (`ee` → plain, `dd`/прочие → TLS)
- Маркировка строк: ✓ живая, ✗ мёртвая
- Сохранение только живых прокси
- Двойной клик по строке — просмотр параметров `server`, `port`, `secret`
- Копирование отдельных параметров в буфер обмена
- Удаление ненужных прокси из результата
- Read-only окно с копированием (Ctrl+C / контекстное меню)
- Кастомная иконка приложения
- Ярлык на рабочем столе для быстрого запуска

## Запуск

```bash
python text_comparer.py
```

Или через ярлык **«TG Proxy»** на рабочем столе (создаётся скриптом `create_shortcut.ps1`).

## Использование

1. Нажмите **«Открыть»** — выберите старый список прокси (File_1), затем новый (File_2)
2. Просмотрите новые прокси, отсутствовавшие в старом списке
3. Выберите метод проверки: **«Быстрая (TCP/TLS)»** (не требует ключей) или **«Точная (Telethon)»** (требует api_id/api_hash)
4. Нажмите **«Проверить»** — прокси маркируются ✓ (жива) / ✗ (мёртва)
5. **Двойной клик** по строке — просмотр параметров server/port/secret
6. Во всплывающем окне: **Copy** — копирование параметра, **Удалить** — исключить прокси
7. Нажмите **«Сохранить»** — сохраняются только живые прокси в File_3
8. **«Закрыть»** — выход из приложения

## Формат строки

```
https://t.me/proxy?server=IP&port=PORT&secret=SECRET
```
